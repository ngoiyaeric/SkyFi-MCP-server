# SkyFi MCP Server

![MCP Server Logo](/img/logo.png)

A **Model Context Protocol (MCP) server** for integrating SkyFi satellite imagery capabilities with AI applications.

## What is SkyFi MCP?

The SkyFi MCP Server bridges the gap between AI applications and satellite imagery, providing seamless access to:

-  **Satellite Image Search & Ordering** - Search and order high-resolution satellite imagery
-  **Archive Data Access** - Access to historical satellite imagery archives  
-  **Real-time Monitoring** - Set up monitoring for specific areas of interest
-  **Multi-source Integration** - Integration with OpenStreetMap and weather data

## Quick Overview

###  What You Get

With SkyFi MCP Server, you get **21 powerful tools** across two main categories:

- **13 SkyFi Tools** for satellite imagery operations
- **8 OpenStreetMap Tools** for geocoding and geospatial analysis

###  Supported AI Clients

- **Claude Desktop** - Full integration with Claude's desktop application
- **Cursor AI** - Enhanced coding with satellite imagery context
- **Windsurf** - Geospatial intelligence in your development workflow  
- **VSCode** - Satellite data access directly in your favorite editor

###  Key Features

- **Easy Setup** - Get started in minutes with any MCP-compatible client
- **Multiple Authentication** - API Key, OAuth 2.0, and Personal Access Token support
- **Docker Ready** - Containerized deployment for production environments
- **Comprehensive Documentation** - Detailed guides, examples, and API reference

Here's what you can accomplish with SkyFi MCP:

```bash
# Ask your AI assistant:
"Search for recent satellite images of Central Park with less than 20% cloud cover"

"Get a pricing quote for satellite tasking over downtown Austin, Texas"
```

Your AI assistant will have access to all 21 tools to help you with satellite imagery tasks, geocoding, and geospatial analysis.

## Architecture Overview

The SkyFi MCP Server acts as a bridge between your AI applications and satellite imagery services, providing a unified interface for complex geospatial operations.

```
AI Client → MCP Protocol → SkyFi MCP Server → SkyFi Platform API
                                         → OpenStreetMap APIs
```
